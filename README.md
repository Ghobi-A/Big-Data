# Big Data Coursework

This repository contains materials for the Big Data coursework submission.

## Setup

1. Create and activate a virtual environment.
2. Install dependencies:

```bash
pip install -r requirements.txt
```

## Folder structure

```text
.
├── data/
│   ├── processed/       # Cleaned or transformed datasets
│   └── raw/             # Original, immutable datasets
├── models/              # Trained model artifacts
├── notebooks/           # Jupyter notebooks (kept in source control)
├── reports/             # Figures and report assets
├── scripts/             # Utility or pipeline scripts
└── src/                 # Reusable source code
```

## Notes

- The coursework notebook lives at the repository root and is unchanged.
